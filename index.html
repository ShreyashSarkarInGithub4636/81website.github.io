Information about 81manorg Linux

Libc: musl v1.2.5

Kernel Version:

Userland: BusyBox, sudo (rewritten in busybox-ash), sbase, util-linux, genfstab (Arch Linux), chroot (Arch Linux, using real-chroot and arch-chroot), udev (Gentoo's eudev), X11 (compiled entirely from scratch).

Window Manager: JWM

Device Manager: udev (Gentoo's eudev)

Shell: sh (BusyBox ash with extended Bash features)

Web Browser: surf

Text Editor: Geany

Initramfs Generator: genbootcpio (written entirely in bash-syntaxed busybox-ash)
